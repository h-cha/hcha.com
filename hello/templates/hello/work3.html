1,作成するソフトウェアの概要

郵便番号を検索すると同じ市区町村の郵便番号と住所のリストが表示される。

同じ市区町村の世代別人口とそれを基にした円グラフが表示される

下の図のようなデータ(postalcode,population)を使った

populationテーブル

2,設計方針

クライアント側はhtmlを読み込むだけでサーバー側はデータベースの実行を行い、行った結果をクライアント側に渡している

index.htmlを実行すると郵便番号検索画面が出る

3,プログラムの説明

String sql = "SELECT * FROM postalcode WHERE code=?";

postalcodeテーブルから一致する郵便番号と住所を表示する

この時得たprefectureとcityで同じ市区町村の郵便番号の一覧を選択する

String sql_code = "SELECT * FROM postalcode WHERE prefecture='" + prefecture + "' and city='" + city + "'";

その後取得したprefectureとciytでpopulationのcityで一致する項目を表示する

例えばpostalcodeのcityは「札幌市中央区」でpopulationのcityは「札幌市　中央区」なので間にスペースを入れないと一致しない

その例外処理を行っている

同じ市区町村の世代別人口を表示
